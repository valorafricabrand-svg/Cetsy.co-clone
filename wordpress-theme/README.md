# Cetsy Blog WordPress Theme

This folder contains a deployable WordPress theme for `https://cetsy.co/blog/`.

## Install

1. Upload `wordpress-theme/cetsy-blog` to the WordPress install at `wp-content/themes/cetsy-blog`.
2. In WordPress Admin, go to Appearance > Themes and activate **Cetsy Blog**.
3. Optional: set a custom logo under Appearance > Customize > Site Identity.
4. Optional: create a Primary menu. If no menu is assigned, the theme still shows the Cetsy marketplace links.

The theme is self-contained and does not require Laravel Vite assets. It uses the Cetsy landing-page fonts, colors, red hero treatment, emerald chips, compact cards, and mobile dock.
